ПРИЗНАЧЕННЯ ЦЬОГО ДОКУМЕНТА

Windows Server 2008 32-бітна та 64-бітна

ОГЛЯД КОМПОНЕНТІВ СИСТЕМИ ДРУКУ

Компакт-диск Windows містить програмне забезпечення для комп'ютерних системи, сумісних з IBM. Для отримання додаткової інформації про певні компоненти драйвера чи програмного забезпечення читайте відповідну інтерактивну довідку.

Драйвери

Драйвер - це програма, яка надає прикладним програмам доступ до функцій пристрою. Як правило, драйвери потрібно встановлювати для кожного виробу. Компакт-диск із програмним забезпеченням для друку, що додається до виробу, містить такі драйвери Windows:

·
драйвер друку HP PCL6 для поданих вище підтримуваних операційних систем;

·
драйвер сканування HP WIA Scan, сумісний з TWAIN.

Програмне забезпечення

·
HP ToolboxFX

·
Центр підтримки та навчання HP LaserJet (надає доступ до посібника користувача виробу та інформації щодо усунення несправностей)

·
Програма видалення

Додаткові драйвери можна отримати у торгового представника Hewlett-Packard або в Інтернеті. Докладнішу інформацію про отримання драйверів та іншого програмного забезпечення HP читайте. у розділі "Технічна підтримка" цього документа або в посібнику користувача виробу.

HP ToolboxFX

Програмне забезпечення для системи друку містить програму HP ToolboxFX. HP ToolboxFX надає інформацію про стан виробу, наприклад про обсяг тонера та паперу, а також відображає попереджувальні повідомлення, якщо робота виробу вимагає певного втручання. HP ToolboxFX можна також використовувати для налаштування параметрів виробу.

Центр підтримки та навчання HP

Центр підтримки та навчання HP встановлюється разом з рекомендованим програним забезпеченням і надає доступ до документації виробу та довідки. Щоб переглянути документацію виробу, не встановлюючи програмного забезпечення, вставте компакт-диск у дисковод для компакт-дисків. Після появи екрана встановлення зліва на екрані натисніть кнопку, що стосується документації.

Програма видалення

Використовуйте програму видалення, щоб видалити компоненти Windows для системи друку HP. Інструкції читайте у розділі "Програма видалення".

II. ВКАЗІВКИ ЗІ ВСТАНОВЛЕННЯ

Програмне забезпечення для налаштування чи встановлення, яке міститься на компакт-диску, визначає версію та мову операційної системи комп'ютера.

ВСТАНОВЛЕННЯ ПРОГРАМНОГО ЗАБЕЗПЕЧЕННЯ

Примітка: Для встановлення системи друку необхідні права адміністратора.

Примітка: Якщо встановлення виконується через USB-підключення, НЕ під'єднуйте кабель USB до виробу чи комп'ютера, доки не з'явиться відповідний запит.

1.
Закрийте усі програми.

2.
Вставте компакт-диск.

3.
Якщо програма встановлення системи друку HP не запустилась через 30 секунд, виконайте кроки 4-6.

4.
Натисніть Start (Пуск) та справа у меню Start (Пуск) виберіть Computer (Комп'ютер) або My Computer (Мій комп'ютер).

5.
Виберіть кореневий каталог на компакт-диску.

6.
Двічі клацніть файл AUTORUN.EXE.

7.
Натисніть кнопку Install (Встановлення), щоб розпочати встановлення програмного забезпечення.

8.
Виконайте вказівки на екрані, щоб завершити встановлення програмного забезпечення.

ПРОГРАМА ВИДАЛЕННЯ

Програма видалення доступна, якщо встановлено рекомендоване програмне забезпечення або програмне забезпечення, яке відповідає мінімальним вимогам. Програма видалення видаляє компоненти програмного забезпечення для системи друку HP.

1.
У меню Start (Пуск) виберіть All Programs (Усі програми) і натисніть HP. За наявності кількох виробів HP виберіть потрібний виріб і натисніть Uninstall from the program group (Видалити з групи програм).

2.
Виконайте вказівки на екрані, щоб видалити програмне забезпечення.

III. НАЙНОВІША ІНФОРМАЦІЯ

ВІДОМІ ПРОБЛЕМИ ТА ЇХ ВИРІШЕННЯ

Загальна інформація

·
Пароль доступу до SNMP можна налаштувати у програмі HP ToolboxFX або на вбудованому веб-сервері. Якщо для пароля доступу встановлено інше значення, аніж "public", пошук мережі припиняється.

·
Роботі програми встановлення HP може перешкоджати брандмауер. Перед встановленням мережі брандмауер слід вимкнути, інакше це може перешкодити виявленню виробу.

·
Деякі плати високошвидкісного порту USB 2.0 постачаються з непідписаними драйверами хост-контролера. Якщо Ви придбали плату розширення USB 2.0 і маєте непідписаний драйвер, можуть виникнути проблеми зі встановленням драйвера або обміном даних між виробом і програмним забезпеченням HP. З обладнанням USB рекомендується завжди використовувати сертифікований драйвер хост-контролера Microsoft WHQL.

·
На комп'ютерах, які відповідають лише мінімальним вимогам до системи, функціональність обмежується. Для достатньої продуктивності та надійності роботи з багатьма видами зображень та відеоматеріалів бажано мати комп'ютер, що відповідає рекомендованим вимогам до системи.

·
Якщо під час встановлення програмного забезпечення виріб не виявлено, перевірте таке:

 
o
Перевірте, чи використовується високошвидкісний кабель USB 2.0.

 
o
Перевірте, чи кабель під'єднаний належним чином. Від'єднайте кабель USB, подивіться на роз'єм на його кінці та перевірте, чи правильно він вставлений в USB-порт. Слідкуйте, щоб роз'єм було обернуто правильною стороною. Вставте один кінець кабелю USB повністю у роз'єм USB на виробі HP, а другий кінець - у відповідний роз'єм позаду комп'ютера.

 
o
Перевірте, чи виріб увімкнений. Якщо виріб вимкнено, програмне забезпечення не зможе його знайти.

 
o
Користуйтеся кабелями USB довжиною не більше 5 м.

 
o
Перевірте, чи увімкнений USB-порт. Деякі комп'ютери продаються з вимкненими USB-портами.

Оновлення системи Windows

Перед оновленням версії системи Windows рекомендуємо видалити програмні продукти HP. Наприклад, оновлюючи Windows Vista до Windows 7, за допомогою програми видалення у групі програм HP видаліть виріб HP (див. розділ "Програма видалення"), після чого виконайте оновлення до нової операційної системи. Після оновлення операційної системи встановіть найновіше програмне забезпечення для пристрою HP, яке можна завантажити на веб-сайті HP.com
