打印系统组件概述

驱动程序

驱动程序是指允许软件应用程序访问产品功能的软件。通常，您应为每个产品安装驱动程序。随设备提供的打印系统软件 CD-ROM 包含以下 Windows 驱动程序：

·
适用于以上列出的受支持操作系统的 HP PCL6 打印驱动程序

·
兼容 TWAIN 的 HP WIA Scan 驱动程序。

软件应用程序

·
HP ToolboxFX

·
HP LaserJet 帮助和学习中心（通过它可访问产品用户指南和故障排除信息）

·
卸载程序

其它驱动程序可从您的 Hewlett-Packard 销售商或从在线资源获得。有关获得驱动程序和其它 HP 软件的更多信息，请参阅此说明文件中的“技术帮助”部分或产品用户指南。

HP ToolboxFX

打印系统软件包括 HP ToolboxFX 软件。HP ToolboxFX 可提供产品状态信息（如碳粉和纸张量）并在产品需要注意时弹出警报消息。您还可以使用 HP ToolboxFX 配置产品设置。

HP 帮助和学习中心

“HP 帮助和学习中心”随推荐软件集安装，通过它可访问产品说明文档和帮助信息。要在不安装推荐软件的情况下查看产品说明文档，请将 CD-ROM 插入 CD-ROM 驱动器。安装程序屏幕出现时，单击屏幕左侧的说明文档按钮。

卸载程序

卸载程序可用于删除 Windows HP 打印系统组件。有关说明，请参阅“卸载程序”部分。

II.安装说明

CD-ROM 设置或安装软件将检测正在运行的操作系统的版本和语言。

安装软件

注意： 安装打印系统需要管理员权限。

注意： 对于 USB 安装，在软件要求之前，请勿将 USB 电缆插入到计算机和产品中。

1.
退出所有程序。

2.
插入 CD-ROM。

3.
如果 30 秒后 HP 打印系统安装程序未启动，则请执行步骤 4 至 6。

4.
单击开始，然后从右侧开始菜单中选择电脑或我的电脑。

5.
浏览并找到 CD-ROM 驱动器的根目录。

6.
双击 AUTORUN.EXE 文件。

7.
单击安装按钮以开始安装软件。

8.
按照屏幕上的说明完成软件安装。

卸载程序

如果安装了推荐软件或最低要求软件，则可使用卸载实用程序。卸载程序将删除 HP 打印系统软件组件。

1.
从开始菜单选择所有程序，然后单击 HP。如果有多个 HP 产品，请选择特定产品，然后从程序组中单击 Uninstall。

2.
按照屏幕上的说明完成软件卸载。

III.最新信息

已知问题和解决办法

一般信息

·
“SNMP 团体名称”可以在 HP ToolboxFX 或内嵌式 Web 服务器中进行设置。如果“团体名称”更改为“public”以外的其它值，则网络扫描操作将停止。

·
防火墙软件可能会影响 HP 软件安装程序的运行。启动网络安装之前，请先禁用防火墙，否则可能检测不到本产品。

·
某些高速 USB 2.0 卡出厂时随附未签名的主控制器驱动程序。如果您购买了带有未签名驱动程序的 USB 2.0 插卡，则可能会遇到驱动程序安装问题，或者设备和 HP 软件之间可能出现通信问题。建议您总是为您的 USB 硬件使用 Microsoft WHQL 认可的主机控制器驱动程序。

·
在仅满足最低系统要求的系统上，功能可能受到一定的限制。大量的图片或视频要求系统必须满足推荐的系统要求，以便获得可接受的性能和可靠性。

·
如果在软件安装期间未检测到产品，请检查以下项目：

 
o
检查电缆是否符合 USB 2.0 高速标准。

 
o
确保电缆连接正确。取下 USB 电缆，检查电缆端头，然后将其与 USB 端口上的图案进行对照比较， 确保上下顺序正确。将 USB 电缆一端完全插入 HP 产品上的 USB 端口，然后将另一端插入计算机背面。

 
o
检查产品是否已打开。如果产品电源关闭，则软件不能发现产品。

 
o
确保您的 USB 电缆长度不超过 5 米（16.4 英尺）。

 
o
检查 USB 端口是否已启用。某些计算机出厂时会禁用 USB 端口。

升级 Windows

在升级到更新的 Windows 支持版本之前，HP 建议您先卸载 HP 产品。例如，从 Windows Vista 升级到 Windows 7 时，请先使用 HP 程序组中的 Uninstaller 程序卸载 HP 产品（参阅“卸载程序”部分），然后再升级到新的操作系统。升级完操作系统后，请使用 HP.com 上提供的最新软件重新安装 HP 产品。
