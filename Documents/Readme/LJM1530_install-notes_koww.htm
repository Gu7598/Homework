인쇄 시스템 구성 요소 개요

Windows CD-ROM에는 IBM 호환 컴퓨터 시스템용 소프트웨어가 들어 있습니다. 특정 드라이버 또는 소프트웨어 구성 요소에 대한 자세한 내용은 관련 온라인 도움말을 참조하십시오.

드라이버

드라이버는 소프트웨어 응용 프로그램을 통해 제품 기능에 액세스할 수 있도록 하는 소프트웨어입니다. 일반적으로 각 제품마다 드라이버를 설치해야 합니다. 장치와 함께 제공되는 인쇄 시스템 소프트웨어 CD-ROM에는 다음과 같은 Windows 드라이버가 포함됩니다.

·
위에 나열된 지원되는 운영 체제용 HP PCL6 인쇄 드라이버

·
TWAIN 호환을 포함하는 HP WIA 스캔 드라이버

소프트웨어 응용 프로그램

·
HP ToolboxFX

·
HP LaserJet 도움말 및 학습 센터(제품 사용 설명서 및 문제 해결 정보 제공)

·
설치 제거 프로그램

추가 드라이버는 Hewlett-Packard 대리점 또는 온라인 소스를 통해 구할 수 있습니다. 드라이버 및 기타 HP 소프트웨어를 구하는 방법은 이 문서의 "기술 지원" 단원 또는 제품 사용 설명서를 참조하십시오.

HP ToolboxFX

인쇄 시스템 소프트웨어에는 HP ToolboxFX 소프트웨어가 포함되어 있습니다. HP ToolboxFX는 토너 및 용지 잔량 등의 제품 상태 정보는 물론, 제품 사용 시 주의가 필요할 때 팝업 경고 메시지를 제공합니다. 또한 HP ToolboxFX를 사용하여 제품 설정을 구성할 수도 있습니다.

HP 도움말 및 학습 센터

HP 도움말 및 학습 센터는 권장 소프트웨어 집합으로 설치되며, 제품 설명서 및 도움말을 이용할 수 있도록 지원합니다. 권장 소프트웨어를 설치하지 않고 제품 설명서를 보려면 CD-ROM을 CD-ROM 드라이브에 넣습니다. 설치 프로그램 화면이 나타나면 해당 화면의 왼쪽에 보이는 설명서 단추를 클릭합니다.

설치 제거 프로그램

설치 제거 프로그램을 사용하여 Windows HP 인쇄 시스템 구성 요소를 제거할 수 있습니다. 자세한 지침은 "설치 제거 프로그램" 단원을 참조하십시오.

II. 설치 지침

CD-ROM 설정 또는 설치 소프트웨어는 실행 중인 운영 체제의 버전과 언어를 자동으로 탐지합니다.

소프트웨어 설치

참고: 인쇄 시스템을 설치하기 위해서는 관리자 권한이 있어야 합니다.

참고: USB 설치의 경우 소프트웨어에서 해당 메시지가 표시될 때까지 USB 케이블을 컴퓨터나 제품에 연결하지 마십시오.

1.
프로그램을 모두 종료합니다.

2.
CD-ROM을 넣습니다.

3.
30초가 지나도 HP 인쇄 시스템 설치 프로그램이 시작되지 않으면 4~6단계를 수행합니다.

4.
시작을 클릭 한 후 오른편에 있는 시작 메뉴에서 컴퓨터 또는 내 컴퓨터를 선택합니다.

5.
CD-ROM 드라이브의 루트 디렉터리로 찾아갑니다.

6.
AUTORUN.EXE 파일을 두 번 누릅니다.

7.
설치 단추를 눌러 소프트웨어 설치를 시작합니다.

8.
화면의 지시에 따라 소프트웨어 설치를 완료합니다.

설치 제거 프로그램

권장 소프트웨어 또는 최소한의 소프트웨어를 설치한 경우 설치 제거 유틸리티를 사용할 수 있습니다. 설치 제거 프로그램을 실행하면 HP 인쇄 시스템의 소프트웨어 구성 요소가 삭제됩니다.

1.
시작 메뉴에서 모든 프로그램, HP를 차례대로 선택하십시오. HP 제품이 둘 이상인 경우, 특정 제품을 선택한 후 프로그램 그룹에서 제거를 클릭합니다.

2.
화면의 지시에 따라 소프트웨어 제거를 완료합니다.

III. 최신 정보

알려진 문제와 해결 방법

일반 사항

·
SNMP 커뮤니티 이름은 HP ToolboxFX 또는 내장 웹 서버에서 설정할 수 있습니다. 커뮤니티 이름이 "public"이 아닌 다른 이름으로 변경되면 네트워크 검색이 중단됩니다.

·
방화벽 소프트웨어는 HP 소프트웨어 설치 프로그램과 충돌을 일으킬 수 있습니다. 네트워크 설치를 시작하기 전에 방화벽을 해제하십시오. 그렇지 않으면 제품이 감지되지 않을 수도 있습니다.

·
특정 고속 USB 2.0 카드는 서명되지 않은 호스트 컨트롤러 드라이버와 함께 제공됩니다. USB 2.0 애드 인 카드를 구입하고 서명되지 않은 드라이버를 보유한 경우, 설치 문제 또는 장치와 HP 소프트웨어 간 통신 문제가 발생할 수 있습니다. 항상 USB 하드웨어와 함께 Microsoft WHQL Certified 호스트 컨트롤러 드라이버를 사용하도록 권장합니다.

·
최소 시스템 요구 사항만을 갖춘 시스템에서는 기능이 제한됩니다. 많은 이미지와 비디오를 사용하는 경우 시스템의 성능 및 안정성 보장을 위해 권장 시스템 요구 사항이 필요합니다.

·
소프트웨어 설치 도중에 제품이 감지되지 않을 경우 다음을 확인하십시오.

 
o
USB 2.0 고속 호환 케이블인지 확인합니다.

 
o
케이블이 올바르게 연결되었는지 확인합니다. USB 케이블을 빼서 케이블의 끝을 확인한 후 USB 포트의 패턴과 일치시킵니다. 케이블이 뒤집혀 있지 않은지 확인합니다. USB 케이블의 한쪽 끝을 HP 제품에 USB 포트에 완전히 연결하고 다른 쪽 끝을 컴퓨터 뒷면에 연결합니다.

 
o
제품이 켜져 있는지 확인합니다. 전원이 꺼져 있을 경우에는 소프트웨어에서 제품을 찾을 수 없습니다.

 
o
USB 케이블의 길이가 5미터를 초과하지 않도록 확인합니다.

 
o
USB 포트가 활성화되어 있는지 확인합니다. 일부 컴퓨터의 경우 USB 포트가 비활성화된 상태로 제공됩니다.

Windows 업그레이드

지원되는 최신 Windows 버전으로 업그레이드하기 전에 HP 제품을 제거하는 것이 좋습니다. 예를 들어, Windows Vista를 Windows 7로 업그레이드할 경우, HP 프로그램 그룹에서 제공한 제거 프로그램을 사용하여 HP 제품을 제거한 다음(설치 제거 프로그램 단원 참조), 새 운영 체제로 업그레이드하십시오. 운영 체제 업그레이드를 마친 후에 HP.com에서 제공되는 최신 소프트웨어를 사용하여 HP 제품을 다시 설치하십시오.
