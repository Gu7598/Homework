Treiber

Ein Treiber ist Software, mit dem Softwareanwendungen auf Produktfunktionen zugreifen können. In der Regel müssen Treiber für jedes Produkt installiert werden. Die CD-ROM mit der Drucksystemsoftware, die mit dem Gerät geliefert wurde, enthält die folgenden Windows-Treiber:

·
HP PCL6-Druckertreiber für die oben genannten unterstützten Betriebssysteme

·
HP WIA-Scantreiber, kompatibel mit TWAIN

Software-Anwendungen

·
HP ToolboxFX

·
HP LaserJet Help and Learn Center (bietet Zugriff auf das Benutzerhandbuch sowie Informationen zur Fehlerbehebung)

·
Deinstallationsprogramm

Zusätzliche Druckertreiber sind eventuell über einen Hewlett-Packard Händler oder Online-Quellen erhältlich. Nähere Informationen über den Bezug von Druckertreibern und anderer HP Software finden Sie im Abschnitt „Technische Unterstützung“ in diesem Dokument oder im Benutzerhandbuch des Produkts.

HP ToolboxFX

Die Drucksystemsoftware beinhaltet die HP ToolboxFX-Software. HP ToolboxFX bietet neben Informationen zum Produktstatus wie Toner- und Papierständen auch Popup-Benachrichtigungen, die angezeigt werden, wenn das Produkt Ihre Aufmerksamkeit erfordert. Sie können HP ToolboxFX auch zum Konfigurieren der Produkteinstellungen verwenden.

HP Help and Learn Center

HP Help and Learn Center wird mit der empfohlenen Softwarekonfiguration installiert und bietet Zugriff auf Produktdokumentation und Hilfe. Wenn Sie auf die Produktdokumentation zugreifen möchten, ohne die empfohlene Software zu installieren, legen Sie die CD-ROM in das CD-ROM-Laufwerk ein. Wenn der Installationsbildschirm angezeigt wird, klicken Sie links auf dem Bildschirm auf die Schaltfläche für die Dokumentation.

Deinstallationsprogramm

Mit dem Deinstallationsprogramm können Sie die HP Drucksystemkomponenten unter Windows entfernen. Weitere Informationen finden Sie im Abschnitt „Deinstallationsprogramm“.

II. INSTALLATIONSANWEISUNGEN

Die Einrichtungs- oder Installationssoftware auf der CD-ROM erkennt die Version und Sprache des ausgeführten Betriebssystems.

INSTALLIEREN DER SOFTWARE

Hinweis: Zur Installation des Drucksystems sind Administratorrechte erforderlich.

Hinweis: Schließen Sie das USB-Kabel erst dann an den Computer oder das Produkt an, wenn die Software Sie dazu auffordert.

1.
Beenden Sie alle Programme.

2.
Legen Sie die CD-ROM ein.

3.
Wenn das Installationsprogramm für das HP Drucksystem nach 30 Sekunden nicht gestartet wird, führen Sie die Schritte 4 bis 6 aus.

4.
Klicken Sie auf Start, und wählen Sie dann Computer oder Arbeitsplatz aus dem Startmenü auf der rechten Seite aus.

5.
Wechseln Sie zum Stammverzeichnis auf dem CD-ROM-Laufwerk.

6.
Doppelklicken Sie auf die Datei AUTORUN.EXE.

7.
Klicken Sie auf die Schaltfläche Installieren, um mit der Installation der Software zu beginnen.

8.
Befolgen Sie die Anweisungen auf dem Bildschirm, um die Installation der Software abzuschließen.

DEINSTALLATIONSPROGRAMM

Das Dienstprogramm für die Deinstallation ist verfügbar, wenn die empfohlene oder minimale Software installiert wurde. Sämtliche Komponenten der HP Drucksystemsoftware werden mit dem Deinstallationsprogramm gelöscht.

1.
Wählen Sie im Menü Start die Option Alle Programme aus, und klicken Sie auf HP. Wenn Sie mehr als ein Produkt von HP verwenden, wählen Sie das jeweilige Produkt aus, und klicken Sie in der Programmgruppe auf Deinstallieren.

2.
Befolgen Sie die Anweisungen auf dem Bildschirm, um die Deinstallation der Software abzuschließen.

III. AKTUELLE INFORMATIONEN

BEKANNTE PROBLEME UND LÖSUNGSANSÄTZE

Allgemein

·
Der SNMP Community Name kann in HP Toolbox FX oder auf dem integrierten Webserver festgelegt werden. Wenn als Community Name etwas anderes als „public“ eingestellt wird, wird die Netzwerksuche gestoppt.

·
Die Firewall-Software kann Konflikte mit dem HP Software-Installationsprogramm auslösen. Deaktivieren Sie vor dem Start einer Netzwerkinstallation die Firewall, da das Produkt andernfalls nicht erkannt wird.

·
Bestimmte hochgeschwindigkeitsfähige USB 2.0-Karten werden mit nicht zertifizierten Host-Controller-Treibern ausgeliefert. Wenn Sie eine USB 2.0-Zusatzkarte erworben haben und über einen nicht zertifizierten Treiber verfügen, können Treiberinstallations- oder Kommunikationsprobleme zwischen dem Gerät und der HP Software auftreten. Es ist empfehlenswert, nur einen Microsoft WHQL-zertifizierten Host Controller-Treiber zusammen mit der USB-Hardware zu verwenden.

·
Auf Systemen, die nur die Mindestvoraussetzungen erfüllen, ist der Funktionsumfang begrenzt. Für viele Bilder und Videos ist ein System erforderlich, das die empfohlenen Systemvoraussetzungen für eine akzeptable Leistung und Zuverlässigkeit erfüllt.

·
Wenn das Produkt bei der Softwareinstallation nicht erkannt wird, überprüfen Sie die folgenden Punkte:

 
o
Prüfen Sie, ob das Kabel für USB 2.0-Hochgeschwindigkeitsdaten geeignet ist.

 
o
Sehen Sie nach, ob das Kabel richtig angeschlossen ist. Entfernen Sie das USB-Kabel und stellen Sie fest, ob das Symbol am Kabelende mit dem Symbol am USB-Anschluss übereinstimmt. Vergewissern Sie sich, dass es korrekt ausgerichtet ist. Stecken Sie das eine Ende des USB-Kabels in den USB-Anschluss am HP Produkt und das andere Ende am Anschluss des Computers ein.

 
o
Überprüfen Sie, ob das Produkt eingeschaltet ist. Nicht eingeschaltete Produkte werden von der Software nicht erkannt.

 
o
Das USB-Kabel darf nicht länger als 5 Meter sein.

 
o
Der USB-Anschluss muss aktiviert sein. Einige Computer werden mit deaktiviertem USB-Anschluss geliefert.

Aktualisieren von Windows
