PRESENTATION DES COMPOSANTS DU SYSTEME D'IMPRESSION

Le CD-ROM Windows contient les logiciels destinés aux systèmes informatiques compatibles IBM. Pour plus d'informations sur un composant spécifique du pilote ou du logiciel, reportez-vous à son aide en ligne.

Pilotes

Un pilote est un logiciel permettant à des applications logicielles d'accéder aux fonctions du produit. Généralement, vous devez installer un pilote pour chaque produit. Le CD-ROM du logiciel du système d'impression livré avec votre périphérique inclut les pilotes Windows suivants :

·
Pilote d'impression HP PCL6 pour les systèmes d'exploitation pris en charge répertoriés ci-dessus

·
Pilote du scanner HP WIA compatible avec TWAIN.

Applications logicielles

·
HP ToolboxFX

·
Centre d'aide et d'apprentissage HP LaserJet (permet d'accéder au guide de l'utilisateur du produit et aux informations de dépannage)

·
Programme de désinstallation

Des pilotes supplémentaires peuvent être disponibles auprès de votre distributeur Hewlett-Packard ou de sources en ligne. Pour plus d'informations sur l'obtention de pilotes et d'autres logiciels HP, reportez-vous à la section « Assistance technique » de ce document ou au guide de l'utilisateur du produit.

HP ToolboxFX

Les logiciels du système d'impression incluent le logiciel HP ToolboxFX. HP ToolboxFX fournit des informations d'état sur le produit, concernant notamment le niveau de toner ou de papier, ainsi que des messages d'alerte contextuels lorsque le produit nécessite une intervention. Vous pouvez également utiliser HP ToolboxFX pour configurer les paramètres du produit.

Centre d'aide et d'apprentissage HP

Le centre d'aide et d'apprentissage HP est installé dans le cadre de l'installation logicielle recommandée et permet d'accéder à la documentation et à l'aide du produit. Pour consulter la documentation de ce produit sans procéder à l'installation logicielle recommandée, insérez le CD-ROM dans le lecteur de CD-ROM. Lorsque l'écran du programme d'installation s'affiche, cliquez sur le bouton Documentation dans la partie gauche de l'écran.

Programme de désinstallation

Le programme de désinstallation permet de supprimer les composants du système d'impression HP pour Windows. Pour plus d'instructions, reportez-vous à la section « Programme de désinstallation ».

II. INSTRUCTIONS RELATIVES A L'INSTALLATION

Le logiciel de configuration ou d'installation du CD-ROM détecte la version et la langue du système d'exploitation que vous exécutez.

INSTALLATION DU LOGICIEL

Remarque : Des privilèges d'administrateur sont requis pour installer le système d'impression.

Remarque : Pour les installations USB, NE branchez PAS le câble USB sur l'ordinateur ou sur le produit avant d'y être invité par le logiciel.

1.
Quittez tous les programmes.

2.
Insérez le CD-ROM.

3.
Si le programme d'installation du système d'impression HP ne démarre pas après 30 secondes, exécutez les étapes 4 à 6.

4.
Cliquez sur Démarrer, puis sélectionnez Ordinateur ou Poste de travail dans la partie droite du menu Démarrer.

5.
Accédez au répertoire racine du lecteur de CD-ROM.

6.
Cliquez deux fois sur le fichier AUTORUN.EXE.

7.
Cliquez sur le bouton Installer pour lancer l'installation du logiciel.

8.
Suivez les instructions à l'écran pour terminer l'installation du logiciel.

PROGRAMME DE DESINSTALLATION

L'utilitaire de désinstallation est disponible si vous avez procédé à l'installation logicielle recommandée ou minimale. Le programme de désinstallation supprime les composants logiciels du système d'impression HP.

1.
Dans le menu Démarrer, cliquez sur Tous les programmes, puis sur HP. Si vous possédez plusieurs produits HP, cliquez sur le produit concerné, puis sélectionnez Désinstaller dans le groupe de programmes.

2.
Suivez les instructions à l'écran pour terminer la désinstallation du logiciel.

III. INFORMATIONS DE DERNIERE MINUTE

PROBLEMES CONNUS ET ALTERNATIVES

Généralités

·
Le nom de communauté SNMP peut être défini dans le logiciel HP ToolboxFX ou dans le serveur Web intégré. Si le nom de communauté est modifié en quelque chose autre que "public", la numérisation réseau cesse de fonctionner.

·
Un logiciel de pare-feu peut interférer avec le programme d'installation de logiciel HP. Avant de démarrer une installation réseau, désactivez le pare-feu, faute de quoi le produit pourrait ne pas être détecté.

·
Certaines cartes USB 2.0 haut débit sont livrées avec des pilotes de contrôleurs d'hôtes non signés. Si vous avez acheté une carte d'extension USB 2.0 et que vous disposez d'un pilote non signé, vous pouvez rencontrer des problèmes d'installation de pilote ou des problèmes de communication entre le périphérique et le logiciel HP. Il est recommandé de toujours utiliser un pilote de contrôleur d'hôte certifié Microsoft WHQL avec votre matériel USB.

·
Le fonctionnement est limité sur les systèmes ne satisfaisant que les exigences système minimales. Une grande quantité d'images et de vidéos requièrent un système respectant les exigences système recommandées pour obtenir des performances et une fiabilité acceptables.

·
Si le produit n'est pas détecté au cours de l'installation du logiciel, vérifiez les éléments suivants :

 
o
Assurez-vous que le câble est bien conçu pour un matériel USB 2.0 haut débit.

 
o
Assurez-vous que le câble est correctement connecté. Retirez le câble USB, examinez son extrémité, puis comparez-la au modèle du port USB. Vérifiez qu'il n'est pas à l'envers. Insérez complètement une extrémité du câble USB dans le port USB du produit HP, puis insérez l'autre extrémité à l'arrière de l'ordinateur.

 
o
Vérifiez que le produit est sous tension. Le produit ne sera pas détecté par le logiciel s'il est hors tension.

 
o
Assurez-vous que la longueur du câble USB ne dépasse pas 5 m.

 
o
Vérifiez que le port USB est activé. Certains ordinateurs sont livrés avec des ports USB désactivés.

Mise à niveau de Windows

HP vous recommande de désinstaller les produits HP avant d'effectuer une mise à niveau vers une version plus récente de Windows. Ainsi, lorsque vous procédez à une mise à niveau de Windows Vista vers Windows 7, désinstallez votre produit HP en utilisant le programme de désinstallation fourni dans le groupe de programmes HP (reportez-vous à la section Programme de désinstallation), puis effectuez la mise à niveau vers le nouveau système d'exploitation. Une fois la mise à niveau du système d'exploitation terminée, réinstallez votre produit HP en utilisant le logiciel le plus récent disponible sur HP.com.
