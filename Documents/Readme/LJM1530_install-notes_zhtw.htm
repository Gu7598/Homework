II.安裝指示

光碟設定或安裝軟體會偵測到您所執行之作業系統的版本和語言。

安裝軟體

注意： 您必須具備管理員權限，才能安裝列印系統。

注意： 若要使用 USB 安裝，除非軟體指示您，否則不要將 USB 纜線插入電腦與產品。

1.
結束所有程式。

2.
插入光碟。

3.
若 HP 列印系統安裝程式在 30 秒後仍未啟動，請完成步驟 4 到 6。

4.
按一下開始，然後從「開始」功能表右側選擇電腦或我的電腦。

5.
瀏覽至光碟機的根目錄。

6.
連按兩下 AUTORUN.EXE 檔案。

7.
按一下安裝按鈕開始進行軟體安裝。

8.
按照螢幕上的說明完成軟體安裝。

解除安裝程式

若已安裝建議的或基本軟體，則會提供解除安裝公用程式。解除安裝程式可移除 HP 列印系統軟體元件。

1.
在開始功能表上，選擇所有程式，然後按一下 HP。若您擁有多個 HP 產品，請選擇指定產品，然後從程式群組中按一下解除安裝。

2.
按照螢幕上的說明完成軟體解除安裝。

III.最新資訊

已知問題與解決方法

一般

·
SNMP 群體名稱可在 HP ToolboxFX 或嵌入式 Web 伺服器中設定。如果「群體名稱」變更為「公用」以外的其他內容，則網路掃描會停止運作。

·
防火牆軟體可能會干擾 HP 軟體安裝程式。請在啟動網路安裝之前停用防火牆，否則可能無法偵測產品。

·
某些高速 USB 2.0 卡含有未簽署的主控制器驅動程式。如果您已購買 USB 2.0 轉接卡，而且有未簽署的驅動程式，可能會遇到驅動程式安裝問題或在裝置和 HP 軟體之間遇到通訊問題。建議您對您的 USB 硬體永遠使用 Microsoft WHQL 認證的主控制器驅動程式。

·
在僅符合最低系統需求的系統上，功能會受到限制。多個影像和影片需要符合建議系統需求的系統，才能獲得可接受的效能和可靠性。

·
如果軟體安裝過程中無法偵測產品，請檢查下列內容：

 
o
確認使用的纜線與高速 USB 2.0 相容。

 
o
確定您的纜線已正確連接。移除 USB 纜線，查看纜線末端，然後比對 USB 連接埠的形式。確定沒有顛倒。將 USB 纜線的一端完全插入 HP 產品上的 USB 連接埠，再把另一端插入電腦背面。

 
o
確認產品電源已開啟。若電源關閉，軟體將無法偵測產品。

 
o
確定您的 USB 纜線長度不超過 5 公尺（16.4 英呎）。

 
o
確認 USB 連接埠已啟用。某些電腦隨附的是停用的 USB 連接埠。

升級 Windows

HP 建議您在升級到支援的 Windows 新版本之前，先解除安裝 HP 產品。例如，從 Windows Vista 升級到 Windows 7 時，請先使用 HP 程式群組（請參閱「解除安裝程式」章節）中的解除安裝程式來為您的 HP 產品解除安裝，然後再將作業系統升級為新版。在完成作業系統升級之後，請至 HP.com 使用最新軟體重新安裝您的 HP 產品
