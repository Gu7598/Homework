DESCRIPCIÓN GENERAL DE LOS COMPONENTES DEL SISTEMA DE IMPRESIÓN

El CD-ROM para Windows contiene software para equipos compatibles con IBM. Para obtener más información sobre un controlador o componente de software concreto, consulte su ayuda en línea asociada.

Controladores

Un controlador es un programa que permite que las aplicaciones de software tengan acceso a las prestaciones del producto. Por lo general deberá instalar controladores para cada producto. El CD-ROM suministrado con el dispositivo, que contiene el software del sistema de impresión, incluye los siguientes controladores para Windows:

·
Controlador de impresión HP PCL6 para los sistemas operativos compatibles indicados anteriormente

·
Controlador de escaneado HP WIA que incorpora compatibilidad TWAIN

Aplicaciones de software

·
HP ToolboxFX

·
Centro de ayuda y formación de HP LaserJet (proporciona acceso a la guía del usuario del producto y a información para la solución de problemas)

·
Desinstalador

Puede solicitar controladores adicionales a su distribuidor de productos Hewlett-Packard o a fuentes en línea. Consulte la sección "Asistencia técnica" de este documento o la guía del usuario del producto para obtener más información acerca de la manera de pedir controladores y otro software de HP.

HP ToolboxFX

El software del sistema de impresión incluye el software HP ToolboxFX. HP ToolboxFX proporciona información sobre el estado del producto, como los niveles de tóner y papel, además de mensajes de alerta que aparecen cuando el producto requiere atención. También puede utilizar HP ToolboxFX para configurar todos los ajustes del producto.

Centro de ayuda y formación de HP

El centro de ayuda y formación de HP se instala junto con la instalación del software recomendado y proporciona acceso a la documentación y la ayuda del producto. Para ver la documentación del producto sin instalar el software recomendado, introduzca el CD-ROM en la unidad de CD-ROM. Cuando aparezca la pantalla del instalador, haga clic en el botón de documentación de la parte izquierda de la pantalla.

Desinstalador

El desinstalador le permite eliminar los componentes del sistema de impresión de HP para Windows. Consulte las instrucciones en la sección "Desinstalador".

II. INSTRUCCIONES PARA LA INSTALACIÓN

El software de configuración o instalación del CD-ROM detecta la versión y el idioma del sistema operativo que se está utilizando.

INSTALACIÓN DEL SOFTWARE

Nota: se necesitan privilegios de administrador para instalar el sistema de impresión.

Nota: en instalaciones con USB, NO conecte el cable USB al equipo o al producto hasta que lo solicite el software.

1.
Cierre todos los programas.

2.
Inserte el CD-ROM.

3.
Si el programa de instalación del sistema de impresión de HP no se inicia al cabo de 30 segundos, realice los pasos 4 a 6.

4.
Haga clic en Inicio y, continuación, seleccione Equipo o Mi PC en el menú de inicio de la derecha.

5.
Desplácese hasta el directorio raíz de la unidad de CD-ROM.

6.
Haga doble clic en el archivo AUTORUN.EXE.

7.
Haga clic en el botón Instalar para iniciar la instalación del software.

8.
Siga las instrucciones que aparecen en pantalla para completar la instalación del software.

DESINSTALADOR

El desinstalador estará disponible si se ha instalado el software mínimo o recomendado. El desinstalador elimina todos los componentes de software del sistema de impresión de HP.

1.
En el menú Inicio, seleccione Todos los programas y, a continuación, haga clic en HP. Si dispone de más de un producto HP, seleccione el producto correspondiente y, a continuación, haga clic en Desinstalar en el grupo de programas.

2.
Siga las instrucciones que aparecen en pantalla para completar la desinstalación del software.

III. INFORMACIÓN DE ÚLTIMA HORA

PROBLEMAS CONOCIDOS Y SOLUCIONES TEMPORALES

Aspectos generales

·
El nombre de comunidad de SNMP se puede definir en HP ToolboxFX o en el servidor Web incorporado. Si se cambia el nombre de comunidad por otro que no sea "public", dejará de funcionar el escaneo en red.

·
El software de firewall puede interferir con el instalador de software de HP. Desactive el firewall antes de iniciar una instalación en red, ya que de lo contrario podría no detectarse el producto.

·
Determinadas tarjetas USB 2.0 de alta velocidad se suministran con controladores de host no firmados. Si ha adquirido una tarjeta USB 2.0 y dispone de un controlador no firmado, tal vez experimente problemas con la instalación del controlador o problemas de comunicación entre el dispositivo y el software de HP. Se recomienda utilizar siempre un controlador de host certificado por Microsoft WHQL con el hardware USB.

·
La funcionalidad es limitada en los sistemas que sólo cumplan los requisitos mínimos. Muchas imágenes y vídeos requieren un sistema que cumpla los requisitos de sistema recomendados con el fin de disfrutar de un rendimiento y una fiabilidad aceptables.

·
Si no se detecta el producto durante la instalación, compruebe lo siguiente:

 
o
Compruebe que el cable es compatible con USB 2.0 de alta velocidad.

 
o
Asegúrese de que el cable está conectado correctamente. Retire el cable USB, examine su extremo y conéctelo al puerto USB adecuado. Asegúrese de que no está conectado al revés. Introduzca un extremo del cable USB en el puerto USB del producto HP y el otro extremo en la parte posterior del equipo.

 
o
Compruebe que el producto está encendido. El software no detectará el producto si está apagado.

 
o
Asegúrese de que la longitud del cable USB no es superior a 5 metros (16,4 pies).

 
o
Compruebe que el puerto USB está activado. Algunos equipos se suministran con los puertos USB desactivados.

Actualización de Windows

HP recomienda que desinstale los productos HP antes de actualizar el sistema operativo a una versión compatible más reciente de Windows. Por ejemplo, si realiza una actualización de Windows Vista a Windows 7, desinstale el producto de HP mediante el programa de desinstalación del grupo de programas de HP (consulte la sección "Desinstalador") y, a continuación, realice la actualización al nuevo sistema operativo. Una vez actualizado el sistema operativo, vuelva a instalar el producto HP con el software más reciente disponible en HP.com
